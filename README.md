# Faq-Accordion-UI
A clean and modern FAQ Accordion built using HTML, CSS, and JavaScript. This project allows users to expand and collapse answers interactively.

live Demo - 


## 🚀 Features
- Smooth accordion toggle effect
- Clean and modern UI design
- Responsive layout
- Lightweight and fast
- Easy to customize

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
